# ROMHub

A browser-based N64 emulator that runs entirely client-side. Upload ROMs from your device, play in browser, save progress locally.

**⚠️ LEGAL DISCLAIMER:**
- This tool requires you to own original game copies
- ROMs are processed locally only - never sent to servers
- Save data persists in your browser's local storage

## Status: DEVELOPMENT

Currently in Phase 0: Emulator core verification.
